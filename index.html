First, you need a function

add = lambda do |x, y|
  x + y
end

add.call(5, 6)

More tersely,

add = -> x, y { x + y }
add[5, 6]

Now, instead of returning the value, wrap the value in a lambda. 

add = lambda do |x, y|
  lambda do 
    x + y
  end
end

add.call(5, 6).call

Now, just move the argument from the outer lambda to the inner lambda! 

add = lambda do |x|
  lambda do |y|
    x + y
  end
end

add.call(5).call(6)
